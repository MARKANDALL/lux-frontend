# public

Static assets served verbatim by Vite. Files here are copied into the build output unchanged and referenced by absolute URL (e.g. `/convo-img/coffee.webp`).

## Contents

- `convo-img/` — conversation scenario thumbnails (generated by `scripts/make-thumbs.mjs`).
- `convo-vid/` — conversation scenario background video loops.
- `vid/` — shared/legacy video assets.
- `assets/characters/` — character portraits used in the Convo picker deck.
- `vendor/` — pinned third-party bundles served locally (WaveSurfer, D3, d3-cloud).
- `lux-popover.js` — small standalone popover helper loaded by a few HTML pages directly.

## Conventions

- Anything here is publicly cached at its absolute path — treat every file as a stable URL.
- Large binaries (videos, vendor bundles) stay pinned here rather than re-installed from npm so the dev/prod asset paths match.
- Regenerate thumbnails with `npm run thumbs` rather than editing `convo-img/` by hand.
