If the retry is still in flight ("Generating…") when the stale timer fires, the label flips back to "Hear it in my voice" mid-request. Low-medium confidence — narrow timing window but reproducible with a slow network.

### INFO

- `features/voice-mirror/voice-mirror.js:592` — Auto-`play()` after render swallows the promise rejection with an empty catch. If browsers block autoplay for this user, the player silently stays paused with no visual hint; consider surfacing that as a one-shot "tap Play" state.

The try/catch swallows it, but the slot can end up orphaned in `#spb-vmMount` until the next open. Same shape as the existing ttsWrap/sourceRow restore code — if you ever revisit this pattern, consider validating `vmSlotNext.parentNode === vmSlotHome` before passing it.
